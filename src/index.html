Важная информация перед использованием сайта

Пожалуйста, прочитайте внимательно

На сайте собраны данные о пунктах выдачи заказов (ПВЗ) от разных служб доставок, чтобы вам было удобнее выбирать. Создатели сайта не несут ответственности за точность информации о пунктах выдачи. Информация собирается из открытых источников. Для каждого ПВЗ указан источник, откуда была взята информация, чтобы вы смогли сами проверить.

Как пользоваться?

  1. Найдите удобный для вас пункт выдачи заказов на карте.
  2. Сканируйте QR-код или переходите по ссылке, чтобы добавить адрес пункта выдачи.
  3. В приложении новый пункт выдачи отобразится как "обл. Ростовская, Пункт Партнёра XXXXXX". Не переживайте – это нормально. Просто сделайте заказ на этот адрес.
  4. Когда получите уведомление о готовности заказа, приходите в выбранный вами пункт выдачи. Чтобы забрать заказ необходимо будет показать штрих-код из приложения.
